Cursor Vibe Coding Tutorial

1. Criar um plano

Antes de fazer qualquer coisa, devemos criar um plano para a ação que desejamos.

Ex:
"
    Você é um especialista em programação e desenvolvimento voltado a streaming de vídeos

    Eu desejo implementar um software que realize streaming de vídeos em meu browser, entretanto não sei como realizar tal tarefa.
    Minha ideia é fazer um streaming similar a Netflix:
    - Um streaming de .mp4 já existente
    - Uma pasta cheia de arquivos .mp4 para que eu escolha o vídeo de desejo

    Contexto:
    * Eu sou um programador experiente em Python
    * Estou acostumado a atuar em projetos backend web em Python

    Tarefa:
    * Me ajude criando um plano para a implementação do software
"

Dessa forma, teremos um direcionamento para nossa produção. Mesmo que a IA traga códigos errados, pelo menos teremos um caminho

2. Definir Ferramentas

Iremos definir, junto a IA, as ferramentas para a produção do sistema

Ex:
"
Considerando a construção anterior, liste as ferramentas de código que eu devo utilizar para a implementação.

Me justifique as escolhas das ferramentas

Regra:
- Seja direto e não faça introdução ou finalização em texto, apenas apresente a listagem com as justificativas
"
